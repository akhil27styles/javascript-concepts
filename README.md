# JavaScript Concepts :mortar_board:

## Description :clipboard:

## Installation :wrench:

1. Clone this repo by running `git clone https://github.com/imranhsayed/javascript-concepts`
2. `cd javascript-concepts`
3. `cd branch-name`
4. `npm install`
5. `npm run server`
